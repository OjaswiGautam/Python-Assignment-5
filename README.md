# Python-Assignment-5

Task 1: Create a Dictionary of Student Marks

This program creates a dictionary where student names are stored as keys and their marks as values. It allows the user to search for a student by name and displays the corresponding marks. If the student is not found, an appropriate message is shown.

Task 2: Demonstrate List Slicing

This program creates a list of numbers from 1 to 10, extracts the first five elements using list slicing, reverses the extracted elements, and displays both the extracted and reversed lists.
